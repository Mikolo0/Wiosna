# Wiosna

Wiosna to drobna aplikacja webowa umożliwiająca użytkownikowi losowanie liczb.

## Instalacja

https://github/INSERTNAME/Wiosna/

## Użycie oraz obsługa

Pobrany folder z plikami należy otworzyć a następnie znaleźć w folderze plik Wiosna.html
a następnie uruchomić go.

Następnie na stronie są przyciski: dodaj, którym można wylosować liczby oraz wyczyść, który opróżnia pola z obecnie wylosowanych liczb.
